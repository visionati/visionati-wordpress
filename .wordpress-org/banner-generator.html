  1. Open this file in Chrome (ensure zoom is 100%)
  2. Open DevTools (F12) → select the .banner-large or .banner-small element
  3. Right-click the element → Capture node screenshot
  4. Rename to banner-1544x500.png or banner-772x250.png and replace the existing files

Tip: On Retina/HiDPI displays, node screenshots capture at device pixel ratio.
For exact 1× dimensions, use DevTools → toggle device toolbar → set DPR to 1.
